# zcmds_win32

Optional zcmds package for win32 to make it feel more like a linux distribution.

# Commands

  * cat
  * cp
  * du
  * git-bash
  * grep
  * home
  * ls
  * mv
  * open
  * rm
  * test
  * touch
  * unzip
  * which


# Install (normal)
  * `python -pip install zcmds`


# Install (dev):

  * `git clone https://github.com/zackees/zcmds_win32`
  * `cd zcmds_win32`
  * `python -pip install -e .`
